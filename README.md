# Mac Dependencies
Assuming you have brew:

```
brew install sdl2
brew install sdl2_ttf
brew install sdl2_image
```

Updated LIBRARY_PATH (to include sdl2 libs):
```
export LIBRARY_PATH="$LIBRARY_PATH:/opt/homebrew/lib"
```
